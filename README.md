# **Modelo de datos:**

### Peluquero:

- Nombre y apellidos
-  nombre de la peluqueria 
-  lugar de la peluqueria
-  valoración(5 estrellas).

### Reservas: 
- Hora de reserva,
- elegir tipo de corte de pelo 
- precio de cada uno.
### Usuarios: 
- ID
- nombre y apellidos
- el historial (incluye la hora, tipo de corte de pelo y el precio).
### Coleccion para trabajos
